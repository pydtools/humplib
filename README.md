# [pytimetools](https://pypi.org/project/drfcommon/#description)
python datetime tools (py开发中的drf common工具集合)

* 分页
* 试图集
* 序列化　

## use in django 
* py3
* django(>=2.2)
* tz:settings.TIME_ZONE

由于目前依赖django, 所以使用py3

```shell
➜  ~ pip install pytimetools

```
